# Installation

Add the theme to your site as a Hugo Module or git submodule under `themes/`:

```bash
git submodule add https://github.com/yilinfang/hugo-researcher.git themes/hugo-researcher
```

Then set `theme = "hugo-researcher"` in your `hugo.toml`. See [`exampleSite/`](../exampleSite)
for a complete, working configuration and content.
